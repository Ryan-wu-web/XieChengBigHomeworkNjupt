## 常见错误码说明

根据 Logcat 中的错误码，可以判断具体问题：

- **1000**: KEY错误 - 检查 Key 配置
- **1001**: KEY无效 - Key 不存在或已删除
- **1002**: KEY过期 - Key 已过期
- **1003**: 服务不可用 - 网络问题或服务维护
- **1004**: 用户KEY未激活 - Key 未启用
- **1005**: 请求KEY与绑定平台不符 - 平台类型错误
- **1006**: 请求KEY与服务不匹配 - 未勾选相应服务
- **1007**: 请求KEY无权限 - Key 权限不足
- **1008**: 安全密钥错误 - 安全密钥配置错误
- **1009**: 平台类型不匹配 - 最常见，平台类型错误
