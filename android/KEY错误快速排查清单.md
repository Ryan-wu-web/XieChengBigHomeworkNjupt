## 常见错误原因

### ❌ 错误 1：SHA1 值不匹配（最常见）

**症状**：KEY错误

**原因**：
- SHA1 值配置错误
- SHA1 值格式错误（缺少冒号或大小写不对）

**解决**：
- 确保 SHA1 值完全一致：`1B:4D:76:7A:D8:C5:72:73:B0:AC:A7:19:13:13:E4:6D:20:13:16:86`
- 注意大小写和冒号

### ❌ 错误 2：包名不匹配

**症状**：KEY错误

**原因**：
- PackageName 配置错误
- 包名有空格或大小写不对

**解决**：
- 确保包名完全一致：`com.yisu.app`
- 不能有空格，不能有大小写差异

### ❌ 错误 3：平台类型错误

**症状**：KEY错误或 USERKEY_PLAT_NOMATCH

**原因**：
- Key 的平台类型不是 Android 平台
- 可能是 Web 端或其他平台的 Key

**解决**：
- 确保 Key 的平台类型是 **"Android 平台"**
- 如果是其他平台，需要重新申请 Android 平台的 Key

### ❌ 错误 4：Key 未生效

**症状**：KEY错误（但配置看起来都正确）

**原因**：
- Key 刚配置完，还没有生效
- 需要等待一段时间

**解决**：
- 等待 **5-30 分钟** 后重试
- 清理并重新构建项目
