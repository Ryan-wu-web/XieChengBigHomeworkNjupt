# 如何打开并运行 Server 和 Vue-Admin

这两个文件夹分别是你项目的**后端（Server）**和**前端（Vue-Admin）**。要让整个系统跑起来，我们需要分别启动它们。

## 1. 运行 Server (后端)
这个项目使用 Java Spring Boot，负责处理数据和逻辑。

1.  **进入目录**: 打开终端，进入 `server` 文件夹。
2.  **检查配置**: 确保 `src/main/resources/application.yml` 中的数据库密码和你电脑上 MySQL 的密码一致（如果你没改过 MySQL 密码且是 `123456` 则不用动）。
3.  **启动命令**: 运行 `mvn spring-boot:run`。
    *   成功后，后端会在 **9090** 端口运行。

## 2. 运行 Vue-Admin (前端)
这个项目使用 Vue 3，是你看到的网页管理界面。

1.  **进入目录**: 打开一个新的终端窗口，进入 `vue-admin` 文件夹。
2.  **安装依赖**: 首次运行需要下载依赖包，运行 `npm install`。
3.  **启动命令**: 运行 `npm run dev`。
    *   成功后，会显示一个访问地址（通常是 `http://localhost:5173`）。

## 执行计划
我将为你执行以下操作：
1.  **启动 Server**: 在一个终端中为你运行后端服务。
2.  **启动 Vue-Admin**: 在另一个终端中为你安装依赖并启动前端页面。
