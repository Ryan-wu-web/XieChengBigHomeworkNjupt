# 后端平台启动与验证方案

你实际上**已经拥有一个完整的后端管理平台架构**。你的项目 `vue-admin` 文件夹就是配套的“后端管理系统前端”，而 `server` 文件夹则提供了完整的管理端 API。

不需要从头开发，我将教你如何**激活并运行**现有的这套系统。

## 1. 现有架构全景图

* **管理端前端**: `vue-admin` (技术栈：Vue 3 + Element Plus + Vite)

  * **功能**: 商家管理、用户管理、酒店上架/审核、房型管理、Banner 广告管理。

* **服务端核心**: `server` (技术栈：Spring Boot + MyBatis-Plus)

  * **功能**: 提供 RBAC 权限控制、数据接口、JWT 认证。

* **数据库**: `MySQL`

  * **数据**: 包含管理员账号 `admin` 和商家账号 `merchant`。

## 2. 启动与验证计划

我们将分三步走，把这个平台跑起来：

### 第一步：启动后端服务 (Server)

* 确保 Spring Boot 后端正在运行（如果你之前关掉了，我会重新启动它）。

* 后端运行在 `http://localhost:9090`。

### 第二步：启动管理端前端 (Vue Admin)

* **安装依赖**: 在 `vue-admin` 目录下运行 `npm install`。

* **启动开发服务器**: 运行 `npm run dev`。

* **访问地址**: 通常是 `http://localhost:5173`。

### 第三步：登录验证

* 使用现有的管理员账号登录：

  * **用户名**: `admin`

  * **密码**: `123456`

* 登录后，你将看到完整的后台管理界面，可以进行酒店管理、用户管理等操作。

## 3. 预期成果

执行完毕后，你将直接拥有一个可用的后台管理系统，无需编数据。
