## 项目经历

### 1. 益宿 (Trip Project) - Android App
**Android 开发负责人 | Java + Retrofit + ViewPager2 + AMap**
- **项目描述**: 酒店预订与管理系统的 Android 客户端，支持酒店搜索、详情展示、地图定位、AI 客服及订单管理。
- **核心贡献**:
    - 基于 **Java + Android 原生组件** 开发，实现高性能酒店预订应用。
    - 使用 **Retrofit + OkHttp** 封装网络层，实现与 SpringBoot 后端的高效交互。
    - 实现 **ViewPager2 + TabLayout** 的 Banner 轮播图与多 Tab 酒店分类展示。
    - 集成 **AMap (高德地图)** SDK，实现城市定位搜索、酒店地理位置展示及周边信息查询。
    - 独立开发 **AI 客服聊天浮动按钮 (FloatingAIButton)** 与 **聊天对话框 (AIChatDialog)**，对接智谱 AI 接口，提供智能预订建议。
    - 实现复杂的酒店列表筛选 UI（价格区间、星级、标签），通过 **ChipGroup** 提供良好的交互体验。
    - 优化 Activity 生命周期管理，确保应用在搜索与详情跳转过程中的性能与稳定性。

### 2. 乡村应用 (微信小程序)
**核心开发者 | 微信小程序 + Vant Weapp + AI 集成**
- **项目描述**: 赋能乡村治理与电商的小程序，涵盖积分申报、AI 审核、随手拍识别及农机预约等功能。
- **核心贡献**:
    - 使用 **Vant Weapp** 组件库构建响应式 UI，确保在微信生态下的流畅体验。
    - 实现“随手拍 AI 识别”模块，通过调用后端 AI 接口实现环境治理问题的自动分类与上报。
    - 负责核心模块（积分申报、电商兑换）的前后端联动开发与接口设计，打通端到端业务流。
    - 优化小程序包体积与启动速度，提升偏远乡村网络环境下用户的访问体验。
