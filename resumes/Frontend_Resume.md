## 项目经历

### 1. 益宿 (Trip Project) - 酒店管理系统
**前端负责人 | Vue 3 + TypeScript + Vite + AMap**
- **项目描述**: 为酒店商家打造的管理后台，支持酒店发布、房间类型管理、用户点评监控及收藏数据看板。
- **核心贡献**:
    - 基于 **Vue 3 + TypeScript + Vite** 构建高性能管理后台，实现模块化开发与类型安全。
    - 集成 **AMap (高德地图)** API，实现酒店地理位置的精准选择与可视化展示，符合携程类 OTA 业务场景。
    - 封装 Axios 请求层，结合 TypeScript 定义后端返回数据模型，提升代码健壮性与开发效率。
    - 实现复杂的酒店列表筛选逻辑（城市、星级、状态），并优化大数据量下的表格渲染性能。
