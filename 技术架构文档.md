# 宜宿酒店预订平台技术架构文档

## 一、项目背景与目标

### 1.1 项目背景
宜宿酒店预订平台是一个面向酒店行业的综合预订管理系统，旨在为酒店商户提供酒店信息发布、管理、审核等服务，同时为消费者提供便捷的酒店查询、预订、收藏、点评等功能。平台集成了AI智能问答功能，提升用户体验。

### 1.2 项目目标
- 构建一个稳定、高效、易用的酒店预订管理系统
- 实现商户自主发布酒店信息，管理员审核的完整流程
- 为消费者提供便捷的酒店搜索、预订、评价服务
- 集成AI智能助手，提供智能问答和推荐服务
- 支持多端访问（Web管理端、Android客户端）

### 3.2 分层架构说明

#### 3.2.1 表现层
- **Android客户端**：提供移动端用户界面，采用MVP架构模式
- **Vue管理端**：提供Web端管理界面，采用Vue 3 + TypeScript开发

#### 3.2.2 控制层
负责接收HTTP请求，参数校验，调用业务逻辑，返回响应结果
- 统一使用`@RestController`注解
- 跨域支持：`@CrossOrigin`
- 统一响应格式：`Result<T>`

#### 3.2.3 业务逻辑层
负责核心业务逻辑处理
- 接口定义：`XxxService`
- 实现类：`XxxServiceImpl`
- 使用MyBatis-Plus提供的CRUD方法

#### 3.2.4 数据访问层
负责数据库操作
- 继承`BaseMapper<T>`
- 支持自定义SQL查询
- MyBatis-Plus提供分页、条件构造器等功能

#### 3.2.5 数据存储层
- MySQL数据库存储业务数据
- 支持事务管理
- 逻辑删除支持

## 五、关键实现逻辑

### 5.1 JWT身份认证

#### 5.1.1 实现原理
```
用户登录 → 验证用户名密码 → 生成JWT Token → 返回客户端
客户端请求 → 携带Token → 服务端验证Token → 执行业务逻辑
```

#### 5.1.2 核心代码
```java
// JwtUtils.java
public class JwtUtils {
    private static final String SECRET_KEY = "yisu-hotel-secret";
    private static final long EXPIRATION = 7 * 24 * 60 * 60 * 1000; // 7天
    
    public static String generateToken(User user) {
        return Jwts.builder()
            .setSubject(user.getUsername())
            .claim("userId", user.getId())
            .claim("role", user.getRole())
            .setExpiration(new Date(System.currentTimeMillis() + EXPIRATION))
            .signWith(SignatureAlgorithm.HS256, SECRET_KEY)
            .compact();
    }
    
    public static Claims parseToken(String token) {
        return Jwts.parser()
            .setSigningKey(SECRET_KEY)
            .parseClaimsJws(token)
            .getBody();
    }
}
```

### 5.2 酒店审核流程

#### 5.2.1 审核状态定义
- 0: 审核中（Auditing）
- 1: 已发布（Published）
- 2: 下线（Offline）
- 3: 审核不通过（Rejected）

#### 5.2.2 审核逻辑
```java
// 商户保存酒店时
@PostMapping("/save")
public Result<Object> save(@RequestBody Hotel hotel) {
    if (hotel.getStatus() != null && hotel.getStatus() == 2) {
        // 下线状态，允许直接设置
    } else {
        // 其他状态强制进入审核流程
        hotel.setStatus(0);
    }
    hotelService.saveOrUpdate(hotel);
    return Result.success();
}

// 管理员审核
@PostMapping("/audit")
public Result<Object> audit(@RequestBody Hotel hotel) {
    Hotel dbHotel = hotelService.getById(hotel.getId());
    dbHotel.setStatus(hotel.getStatus());
    if(hotel.getStatus() == 3) {
        dbHotel.setRejectReason(hotel.getRejectReason());
    }
    hotelService.updateById(dbHotel);
    return Result.success();
}
```

### 5.3 分页查询

#### 5.3.1 实现方式
使用MyBatis-Plus的`Page`对象和`QueryWrapper`条件构造器

```java
@GetMapping("/list")
public Result<Page<Map<String, Object>>> list(
    @RequestParam(defaultValue = "1") Integer pageNum,
    @RequestParam(defaultValue = "10") Integer pageSize,
    @RequestParam(required = false) String name,
    @RequestParam(required = false) String city) {
    
    Page<Hotel> page = new Page<>(pageNum, pageSize);
    QueryWrapper<Hotel> queryWrapper = new QueryWrapper<>();
    queryWrapper.eq("status", 1); // 只查询已发布的酒店
    
    if (StrUtil.isNotBlank(name)) {
        queryWrapper.like("name", name);
    }
    if (StrUtil.isNotBlank(city)) {
        queryWrapper.eq("city", city);
    }
    
    Page<Hotel> hotelPage = hotelService.page(page, queryWrapper);
    return Result.success(hotelPage);
}
```

```java
public Result<String> upload(@RequestParam("file") MultipartFile file) {
    if (file.isEmpty()) {
        return Result.error("文件不能为空");
    }
    
    String fileName = file.getOriginalFilename();
    String filePath = "uploads/" + fileName;
    
    File dest = new File(filePath);
    if (!dest.getParentFile().exists()) {
        dest.getParentFile().mkdirs();
    }
    
    file.transferTo(dest);
    return Result.success(filePath);
}
```

#### 5.5.2 对话实现

```java
@Service
public class AIServiceImpl implements AIService {
    
    @Value("${ai.zhipu.api-key}")
    private String apiKey;
    
    @Value("${ai.zhipu.model}")
    private String model;
    
    @Override
    public String chat(String message, boolean hotelQuery) {
        OkHttpClient client = new OkHttpClient.Builder()
            .connectTimeout(30, TimeUnit.SECONDS)
            .readTimeout(30, TimeUnit.SECONDS)
            .build();
        
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", model);
        
        JSONArray messages = new JSONArray();
        JSONObject systemMessage = new JSONObject();
        systemMessage.put("role", "system");
        systemMessage.put("content", hotelQuery ? 
            "你是一个酒店预订助手..." : 
            "你是一个智能助手...");
        messages.add(systemMessage);
        
        JSONObject userMessage = new JSONObject();
        userMessage.put("role", "user");
        userMessage.put("content", message);
        messages.add(userMessage);
        
        requestBody.put("messages", messages);
        
        Request request = new Request.Builder()
            .url("https://open.bigmodel.cn/api/paas/v4/chat/completions")
            .addHeader("Authorization", "Bearer " + apiKey)
            .post(RequestBody.create(
                requestBody.toJSONString(),
                MediaType.parse("application/json")))
            .build();
        
        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                JSONObject result = JSON.parseObject(responseBody);
                return result.getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");
            }
        }
        return "抱歉，我暂时无法回答您的问题。";
    }
}
```
